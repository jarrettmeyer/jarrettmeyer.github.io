# jarrettmeyer.github.io

## Package Manager

**Always** use `bun`. **Never** use `npm`, `yarn`, or `pnpm`.

## Superpowers

When using the Superpowers skill, write all outputs to `.claude/superpowers/`.

- **Brainstorming**: `.claude/superpowers/brainstorm/`
- **Specs**: `.claude/superpowers/specs`

### Visual Companion

You are allowed to show visual suggestions (mockups, diagrams, comparisons) in the browser companion without asking for permission first. Just start the server and show the content.
