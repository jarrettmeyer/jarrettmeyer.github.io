## Solution

<LeetCodeResult runtime={15} beats={80.95} />

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        best = 0
        chars = []
        for char in s:
            if char in chars:
                chars = chars[chars.index(char) + 1:]
            chars.append(char)
            if len(chars) > best:
                best = len(chars)
        return best
```
