## Solution

<LeetCodeResult runtime={0} beats={100.0} />

```python
class Solution:
    def kidsWithCandies(self, candies: List[int], extraCandies: int) -> List[bool]:
        answer = []
        most = max(*candies)

        for i in range(len(candies)):
            if candies[i] + extraCandies >= most:
                answer.append(True)
            else:
                answer.append(False)

        return answer
```
